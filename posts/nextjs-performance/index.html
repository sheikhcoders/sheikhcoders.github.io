Starter Guide to Next.js Performance

Embrace the app/ router for automatic code-splitting and metadata APIs, then optimize fonts and images.

app/ রাউটার ব্যবহার করলে স্বয়ংক্রিয় কোড স্প্লিটিং ও মেটাডাটা এপিআই পাওয়া যায়—সাথে ফন্ট ও ইমেজ অপ্টিমাইজ করুন।

  • Next.js
  • Core Web Vitals
  • Meta Tags

Add next/font for critical font inlining and use the Image component for responsive assets. Bundle analyze early to spot bloated dependencies.

  • Enable experimental.optimizePackageImports to trim unused library code in production.
  • Measure Core Web Vitals via reportWebVitals and push results to your preferred analytics endpoint.
  • Create a Lighthouse CI budget (largest contentful paint < 2.5s, total blocking time < 200ms) before launch.

next/font দিয়ে ক্রিটিকাল ফন্ট ইনলাইন করুন এবং Image কম্পোনেন্ট ব্যবহার করে রেস্পন্সিভ ইমেজ পরিবেশন করুন। শুরুতেই বান্ডল বিশ্লেষণ করলে অপ্রয়োজনীয় ডিপেন্ডেন্সি ধরা পড়ে।

  • প্রোডাকশনে অব্যবহৃত লাইব্রেরি কোড কমাতে experimental.optimizePackageImports সক্রিয় করুন।
  • reportWebVitals ব্যবহার করে কোর ওয়েব ভাইটালস মাপুন এবং পছন্দের অ্যানালিটিক্স এন্ডপয়েন্টে পাঠান।
  • লাইভ করার আগে লাইটহাউস সি আই বাজেট সেট করুন (LCP < ২.৫ সেকেন্ড, TBT < ২০০ মিলিসেকেন্ড)।
